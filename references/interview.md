# 访谈问题库与追问策略

分批提问（每轮 ≤3 问），根据回答动态追问。以下按维度组织，`→` 表示触发追问。

## 1. 体态问题

- 「你觉得自己有哪些体态问题？（圆肩驼背 / 头前引 / 骨盆前倾或后倾 / 高低肩 / 膝内扣或 O 型腿 / 扁平足 / 说不清楚）」
- 说不清楚 → 给自查方法：
  - **靠墙测试**：脚跟、臀、上背贴墙，后脑勺够不到墙 = 头前引/驼背倾向
  - **侧面照片**：耳垂-肩峰-髋是否一条垂线；腰椎前凸是否明显（骨盆前倾）
  - **正面照片**：双肩是否等高、锁骨是否对称
- 久坐人群追问：每天坐多久？工作姿势（电脑高度、是否翘腿）？
- → 记录：主要问题（排序）、左右不对称细节

## 2. 疼痛与病史（红旗筛查，必问）

- 「现在身体有没有哪里疼？什么情况下疼？」（部位 / 静息痛还是动作痛 / 持续多久了）
- 「有没有确诊过的骨科问题或做过手术？」（椎间盘、肩袖、半月板、踝扭伤史…）

**红旗信号 → 停止生成激进计划，明确建议先就医评估**：
- 急性疼痛（< 2 周）或夜间静息痛
- 放射性疼痛/麻木（如坐骨神经症状、手臂放电感）
- 确诊未愈的结构性损伤
- 不明原因的关节肿胀

非红旗的慢性劳损 → 计划中该部位只做低负荷康复级动作，tip 写明停止条件，并建议同步就医确认。

## 3. 目标

- 「最想解决什么？」体态矫正 / 增肌 / 减脂 / 体能 / 建立运动习惯（多选则排优先级）
- 「有没有具体的标志性目标？」（如：完成第一个引体、体脂降到 X、不再腰酸）
- 目标与条件冲突时（如每周 2 天想大幅增肌）→ 直说预期需要调整

## 4. 训练条件

- 「每周能稳定拿出几天训练？每次多长时间？」（宁少勿虚报，鼓励保守承诺）
- 「在哪练？有什么器械？」徒手在家 / 弹力带+哑铃 / 小区单杠 / 商业健身房
- 「早上还是晚上练？」（影响晨间矫正模板的编排建议）

## 5. 经验水平

- 「有规律训练的经历吗？多久？」
- 「深蹲、硬拉、引体、俯卧撑这些动作熟悉哪些？」
- 零基础 → 计划从自重/弹力带感知动作开始，前 2 周刻意低量
- 有经验 → 询问当前大致负荷水平，起始重量给区间让用户第一周自测

## 6. 计划长度与节奏

- 「想要多长的计划？」4 周（入门试水）/ 8 周（默认推荐）/ 12 周（完整周期）
- ≥8 周 → 默认每 4 周一个减量周，问用户是否接受
- 「习惯打卡想跟踪什么？」（睡眠 / 饮水 / 蛋白质 / 久坐起身 / 拉伸——挑 4-8 个，宁少勿多）

## 访谈摘要模板

```
📋 访谈摘要
体态问题: …（按优先级）
疼痛/病史: …（含处理策略）
目标: …
条件: 每周 X 天 × Y 分钟,器械: …
经验: …
计划: X 周,Phase 划分: …,减量周: …
习惯跟踪: …
体测指标: …
```

用户确认摘要后再开始生成 plan.js。
